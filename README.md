# 📂 Desafios de Frontend - HTML & CSS

Este repositório foi criado para centralizar todos os meus projetos e desafios práticos desenvolvidos para consolidar os meus conhecimentos em HTML5, CSS3 e design responsivo. O meu objetivo principal é colocar em prática os conceitos teóricos aprendidos no **freeCodeCamp** e criar interfaces reais a partir de designs profissionais.

## 🚀 Tecnologias Utilizadas
* HTML5 (Semântico)
* CSS3 (Flexbox, Grid, Variáveis e Animações)
* Git & GitHub (Controlo de versões)
* GitHub Pages (Hospedagem automatizada)

## 🛠️ Como Executar os Projetos Localmente

Se quiseres testar qualquer um dos projetos na tua máquina local, segue estes passos:

1. Clona este repositório:
   ```bash
   git clone https://github.com
   ```
2. Entra na pasta do repositório:
   ```bash
   cd desafios-frontend
   ```
3. Abre a pasta do desafio desejado e clica duas vezes no ficheiro `index.html` para abrir diretamente no teu navegador.
